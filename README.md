# ACFrontEnd

Sistema Front-End desenvolvido para gerenciamento e acompanhamento de serviços, utilizando interface web moderna, responsiva e organizada para facilitar o controle das informações e melhorar a experiência do usuário.

---

## 📌 Funcionalidades

- Cadastro de serviços
- Listagem de informações em tempo real
- Interface responsiva
- Navegação dinâmica
- Organização de dados
- Componentização utilizando React
- Comunicação com API
- Gerenciamento de estados da aplicação

---

## 🚀 Tecnologias Utilizadas

- React.js
- JavaScript
- HTML5
- CSS3
- Node.js
- Git e GitHub

---


## ⚙️ Como Executar o Projeto

Clone o repositório:

```bash
git clone https://github.com/victords97/ACFrontEnd.git

Acesse a pasta do projeto:

cd ACFrontEnd

Instale as dependências:

npm install

Execute o projeto:

npm start

O sistema estará disponível em:

http://localhost:3000
📖 Objetivo do Projeto

O projeto foi desenvolvido com o objetivo de criar uma interface web moderna e funcional para gerenciamento de serviços, permitindo melhor organização, visualização e controle das informações através de uma aplicação prática e intuitiva.
